Adding a Cloud Cluster#

  1. Click "Request cloud cluster".

  2. Enter a name for your cloud cluster and click "Confirm". (The other fields can not yet be altered.)

Editing a Cloud Cluster top#

To edit a cloud cluster, click the three-dotted menu at the top right corner of a cluster card, then click "Edit". You can then rename your cloud cluster. (The other fields can not yet be altered.)

Deleting a Cloud Cluster top#

To delete a cloud cluster, click the three-dotted menu then click "Delete".

Autopilot for Cloud cluster top#

  • Autopilot: Values supported by cloud clusters and the vehicle bridge will be listed after clicking the field. Click "Show More" to view other possible values.
  • Docker Image: The field is filled in with the default Autopilot image supplied by the web user interface. You can use it, or enter the NAME:TAG of an image you have published to a publicly accessible Docker registry. Press the refresh button (arrows in a circle) to return the field to its default.

See here for information on how to build and publish your own Apollo 5.0 image.
